# Language Detection using Machine Learning:-

# Project Overview:-
This project builds a Machine Learning model to automatically detect the language of a given text input.
The system uses Natural Language Processing (NLP) techniques to preprocess text and classify it into different languages.

# Problem Statement
With the rapid growth of multilingual content on the internet, automatic language detection is essential for:

- Content moderation
- Chatbots
- Translation systems
- Search engines
- Social media filtering
This project aims to build a robust language classification model.
